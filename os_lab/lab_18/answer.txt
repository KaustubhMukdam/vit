How program & data cards are stored in memory

Program cards: Loaded sequentially starting at memory address M[0]. Each memory cell M[i] holds 4 characters; program text is written into successive M[0], M[1], … and the loader advances the pointer m in 10-word (block) boundaries (blocks of 10 memory words).
Data cards: On a GD instruction the kernel reads a data card into the memory block whose starting address is the instruction operand. The code writes up to 10 memory words starting at M[operand] (i.e., M[10]..M[19] for operand 10), placing 4 chars per M cell and padding with * for unused bytes.
